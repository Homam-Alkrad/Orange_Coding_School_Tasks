CSS Properties Explained

Position: Relative

The `position: relative;` property positions an element relative to its normal position. It allows you to move an element using the `top`, `right`, `bottom`, and `left` properties, but it still takes up space in the document flow.

This box is positioned relatively with top: 20px, left: 20px.

Position: Absolute

The `position: absolute;` property positions an element relative to its closest positioned ancestor. If none is found, it uses the document body, and the element is removed from the normal document flow.

This box is positioned absolutely inside its container.

Color

The `color` property sets the color of the text content in an element. You can specify the color using a color name, hex value, RGB, or HSL.

This text is red using the `color: red;` property.

This text is blue using the `color: #3498db;` property.

CSS Grid

The `display: grid;` property creates a grid-based layout system. You can define rows, columns, and areas within the grid, making it a powerful tool for creating complex layouts.

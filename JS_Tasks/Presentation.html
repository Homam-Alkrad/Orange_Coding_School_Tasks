Understanding JavaScript Loops and Functions

1. What is the for Statement in JavaScript?

The for loop in JavaScript is used when you know the number of iterations beforehand. It consists of three parts: initialization, condition, and increment. It repeats a block of code as long as the specified condition is true.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

2. What is the while Statement in JavaScript?

The while loop continues to execute a block of code as long as the condition is true. It is often used when the number of iterations is not known beforehand.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

3. What is the do-while Statement in JavaScript?

The do-while loop is similar to the while loop, but it guarantees that the block of code will be executed at least once before checking the condition.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

4. What is a function in JavaScript?

A function is a block of code designed to perform a particular task. It is executed when it is invoked (called). Functions can take inputs (parameters) and return outputs.

function greet(name) {
    return 'Hello, ' + name;
}

console.log(greet('Homam'));

Differences Between for, while, and do-while

The main difference between these loops is how they check the condition:

  • for: Best when you know the exact number of iterations.
  • while: Best when the number of iterations is unknown, and you need to keep checking the condition.
  • do-while: Best when you need the loop to execute at least once, regardless of the condition.

JavaScript map() Method

Learn how to use the map() method with various parameters and real-world object data.

Example 1: Basic Transformation

Transform each element by doubling its value.


const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6]

Example 1: Basic Transformation

Transform each element by doubling its value.


const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6]

Example 2: Using Index in map()

Add the index to each element's value.


const numbers = [10, 20, 30];
const indexed = numbers.map((num, index) => num + index);
console.log(indexed); // Output: [10, 21, 32]

Example 3: Accessing the Entire Array in map()

Use the original array to access other elements while processing.


const numbers = [1, 2, 3];
const summedWithArray = numbers.map((num, index, arr) => num + arr[0]);
console.log(summedWithArray); // Output: [2, 3, 4]

Example 4: Processing Object Data

Apply a 10% discount on product prices.


const products = [
  { name: 'Laptop', price: 1000 },
  { name: 'Phone', price: 500 },
  { name: 'Tablet', price: 300 }
];

const discounted = products.map(product => ({
  name: product.name,
  discountedPrice: product.price * 0.9
}));
console.log(discounted);

Example 5: Mapping Object Properties

Extract just the names from an array of user objects.


const users = [
  { name: 'Homam', age: 25 },
  { name: 'Ahmad', age: 22 },
  { name: 'Osama', age: 30 }
];

const userNames = users.map(user => user.name);
console.log(userNames); // Output: ['Homam', 'Ahmad', 'Osama']

Example 6: Performing Mathematical Operations

Double each number in the array.


const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(number => number * 2);
console.log(doubled); // Output: [2, 4, 6, 8, 10]

Example 7: Creating New Structures from Objects

Convert student data into a pass/fail summary based on scores.


const students = [
  { name: 'Homam', score: 85 },
  { name: 'Ahmad', score: 75 },
  { name: 'Osama', score: 90 }
];

const passFailSummary = students.map(student => ({
  name: student.name,
  passed: student.score >= 80
}));
console.log(passFailSummary);

Example 8: Mapping DOM Elements

Extract the text content of all paragraph elements on the page.


const paragraphs = Array.from(document.querySelectorAll('p'));
const paragraphTexts = paragraphs.map(p => p.textContent);
console.log(paragraphTexts);

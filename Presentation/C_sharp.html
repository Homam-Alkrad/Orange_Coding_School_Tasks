C# Data Types

C# offers several data types, such as:

  • int: integer numbers
  • double: floating-point numbers
  • bool: true or false
  • string: text

1. Casting Types

You can cast between compatible data types. There are two types of casting: implicit (automatic) and explicit (manual).

1. Explicit Casting

Explicit casting is needed when converting from a larger type to a smaller type, as there is potential for data loss:

double d = 9.8;
int i = (int)d; // Explicit casting

Output: i is now 9 (decimal part is truncated).

2. Implicit Casting

Implicit casting is done automatically when converting from a smaller type to a larger type (e.g., int to double):

int num = 10;
double d = num; // Implicit casting (int to double)

Output: d is now 10.0 (integer is promoted to a floating-point).

3. Using Convert Methods

You can also use the Convert class to cast types, especially for converting between incompatible types:

string str = "123";
int num = Convert.ToInt32(str); // Using Convert class

Output: num becomes 123 (converted from a string).

4. Casting Between Numeric Types

You can cast between other numeric types, such as float, double, and long:

float f = 5.75F;
double d = f; // Implicit casting (float to double)
long l = (long)f; // Explicit casting (float to long)

Output: d is 5.75, and l is 5 (fractional part removed).

5. Casting Between Non-Primitive Types

You can also cast between non-primitive types, such as objects:

object obj = "Hello";
string str = (string)obj; // Casting an object to a string

Output: str is now "Hello".

6. Handling Invalid Casts

Be cautious when casting. If the types are incompatible, an exception will occur. For instance:

string str = "abc";
int num = Convert.ToInt32(str); // Throws an exception: FormatException

Note: Always ensure the type you're casting is compatible, or use try-catch for safe conversions.

2. Using Conversion Methods

C# has built-in methods for converting data types:

int num = Convert.ToInt32("123");

Output: num becomes 123 (converted from a string).

7. User Input (Read and Write)

To read user input, use Console.ReadLine() and to output, use Console.WriteLine():

Console.WriteLine("Enter your name:");
string name = Console.ReadLine();
Console.WriteLine("Hello, " + name);

Output: Greets the user by name.

Using String Format

You can also format strings using String Format to make your output more readable:

Console.WriteLine("Enter your age:");
int age = Convert.ToInt32(Console.ReadLine());
Console.WriteLine("Hello {0}, you are {1} years old.", name, age);

Output: Outputs: Hello [name], you are [age] years old.

Using String Interpolation (with $)

String interpolation allows you to include variables directly inside a string with the $ symbol:

Console.WriteLine($"Hello {name}, you are {age} years old.");

Output: Outputs the same result as String Format, but with simpler syntax: Hello [name], you are [age] years old.

3. Arithmetic Operators

C# supports basic arithmetic operators:

  • Addition: a + b
  • Subtraction: a - b
  • Multiplication: a * b
  • Division: a / b
int sum = 5 + 3; // Output: 8

4. Assignment Operators

Assignment operators are used to assign values:

int x = 5;
x += 2;  // Output: 7

10. Comparison and Logical Operators

Comparison operators compare values, and logical operators combine conditions:

  • Equal: a == b
  • Not equal: a != b
  • And: a && b
  • Or: a || b
bool result = (5 == 5) && (3 < 4);  // Output: true

5. Declaring and Initializing an Array

To declare an array in C#, specify the type followed by square brackets:

int[] numbers;

This only declares the array but does not initialize it. To initialize it, you can assign values like so:

numbers = new int[] { 1, 2, 3, 4, 5 };

Output: An array of integers initialized with values 1 to 5.

You can also declare and initialize the array in one line:

int[] numbers = { 1, 2, 3, 4, 5 };

User-Defined Array Size

You can ask the user to define the size of the array and then populate it:

Console.WriteLine("Enter the size of the array:");
int size = Convert.ToInt32(Console.ReadLine());
int[] userArray = new int[size];

This initializes an array where the size is determined by the user.

Filling the Array

Once the array is declared with a user-defined size, you can fill it with values:

for (int i = 0; i < size; i++) {
    Console.WriteLine($"Enter value for index {i}:");
    userArray[i] = Convert.ToInt32(Console.ReadLine());
}

Output: The array is filled with the user-entered values.

Displaying the Array

After the array is filled, you can display its values:

Console.WriteLine("Your array contains:");
for (int i = 0; i < size; i++) {
    Console.WriteLine(userArray[i]);
}

Output: Displays the elements of the user-defined array.

7. Accessing Array Elements

To access an element, use the index of the array:

int firstElement = numbers[0];

Output: The first element is 1 (index starts from 0).

3. Changing Array Elements

You can change an element by assigning a new value:

numbers[2] = 10;

Output: The third element (index 2) becomes 10.

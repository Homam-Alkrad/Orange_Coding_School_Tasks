JavaScript Objects and Arrays

Object in JavaScript

An object is a collection of properties, where each property is defined by a key-value pair. Example:


let person = {
    firstName: 'John',
    lastName: 'Doe',
    age: 30
};

Arrays in JavaScript

Arrays are a collection of elements that can be of any type. Here's an array:

let fruits = ['Apple', 'Banana', 'Orange'];

Array Methods

Comparison of Array Methods

Table 1: Differences between shift(), pop(), and delete()

Method Description Effect on Array
shift() Removes the first element of the array and shifts the remaining elements to the left. The length of the array is reduced by 1. No gaps.
pop() Removes the last element of the array. The length of the array is reduced by 1. No gaps.
delete Removes the element at a specific index but leaves undefined in its place. The length of the array remains the same, but there may be empty slots in the array.

Table 2: Differences between shift() and push()

Method Description Effect on Array
shift() Removes the first element of the array and shifts the remaining elements to the left. The length of the array is reduced by 1. The first element is removed.
push() Adds one or more elements to the end of the array. The length of the array is increased by the number of elements added. New elements are added at the end.

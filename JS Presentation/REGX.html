Understanding Regular Expressions (Regex)

What is a Regular Expression?

A regular expression (regex) is a sequence of characters that defines a search pattern. It is primarily used for string pattern matching, validation, and text manipulation in various programming languages.

Common Regex Patterns

Here are a few commonly used regular expressions:

1. Validate an email:
/^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$/
Explanation: Matches a standard email format.
2. Validate a phone number:
/^\d{3}-\d{3}-\d{4}$/
Explanation: Matches a phone number in the format 123-456-7890.
3. Validate a URL:
/^(https?:\/\/)?([\w\-])+\.{1}[a-zA-Z]{2,}(\/[\w\-]*)*$/
Explanation: Matches a URL, including optional http/https.
4. Validate a date (YYYY-MM-DD):
/^\d{4}-\d{2}-\d{2}$/
Explanation: Matches dates in the format YYYY-MM-DD (e.g., 2023-12-31).
5. Match alphanumeric characters:
/^[a-zA-Z0-9]+$/
Explanation: Matches strings containing only letters and numbers.
6. Validate a strong password:
/^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[\W_]).{8,}$/
Explanation: Requires at least 8 characters, including a lowercase letter, uppercase letter, number, and special character.
7. Match hexadecimal color codes:
/^#([A-Fa-f0-9]{6}|[A-Fa-f0-9]{3})$/
Explanation: Matches a 3- or 6-digit hexadecimal color code (e.g., #FFF or #123ABC).
